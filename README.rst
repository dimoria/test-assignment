# Module Control Work – NumberList

## Автор
- **ПІБ:** Дубенок Діана  
- **Група:** (ІM-34)  
- **Номер у списку групи:** 4  

## Персональні параметри завдання
- Номер у списку групи: **4**
- **C3 = 1** → кільцевий однонаправлений список  
- **C5 = 4** → шістнадцяткова система числення  
- Додаткова система числення: **двійкова**
- **C7 = 4** → операція остачі від ділення (mod)

## Реалізація
- Тип списку: **кільцевий однонаправлений**
- Внутрішня структура: власний клас `Node`
- Тип елемента: `byte`
- Основна система числення: **16**
- Додаткова система числення: **2**
- Додаткова операція: **залишок від ділення двох чисел**
- Операнди не змінюються, результат повертається у новому списку

## Збірка та запуск тестів

### Вимоги
- **Java JDK 17+**
- **GNU Make**

### Доступні команди

Показати список доступних команд:
```bash
make help
```
Завантажити залежності (JUnit):
```bash
make deps
```
Скомпілювати основний код і тести:
```bash
make compile
```
Запустити всі тести:
```bash
make test
```
Очистити результати збірки:
```bash
make clean
```
